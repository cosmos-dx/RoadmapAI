# RoadMapAI
RoadMapAI is an AI agent that creates roadmaps for various tech-related fields. It provides two types of roadmaps: Basic and Full.

Basic Roadmap: Outlines the essential skills and steps required.
Full Roadmap: Generates a detailed step-by-step guide with insights from the Career Advisory Agent and the Quality Assurance Agent.


### env variables 
```
SERPER API
GEMINI API
GOOGLE API 
```

### Query

```
{
	query: "Provide a RoadMap to learn Golang. I know the basic syntax and methods.
	type: "full"
}
```

### How to run locally ?

``` 
git clone https://github.com/cosmos-dx/RoadmapAI.git
pip install -r requirements.txt
python app.py
```
